# Custom i3wm & Polybar Configuration

A lightweight, high-performance desktop setup for i3wm and Polybar, specifically optimized for systems with limited hardware resources (like Core 2 Duo / 3GB RAM). 

This configuration focuses on a minimalist aesthetic without compromising on productivity, featuring a clean look with sharp corners and no heavy animations.

## ✨ Features
- **i3-wm**: Fast, tiled window management.
- **Polybar**: Customized status bar with system monitoring (CPU, RAM, Network).
- **Picom**: Optimized for performance (Animations and Blur disabled).
- **Theme**: Dark aesthetic (Dracula-inspired).
- **Font**: JetBrainsMono Nerd Font.

## 🛠️ Prerequisites

Before installing, ensure you have the following packages installed on your system (Arch/Manjaro based):

```bash
sudo pacman -S i3-wm polybar picom feh rofi ttf-jetbrains-mono-nerd
